# MacroLog

A self-contained calorie and macro tracking PWA. No build step, no frameworks — a single `index.html` with vanilla JS, storing everything locally in the browser (`localStorage`).

Live app: https://REPLACE_WITH_USERNAME.github.io/macrolog/

## Features
- Dashboard with calorie/macro progress bars and a food log grouped by time of day
- Food search (USDA FoodData Central + Open Food Facts), history, favorites, and guided wizards for common foods (steak, chicken, fish, pork, eggs, pasta)
- Optional AI macro estimation from a text description or photo (bring your own Claude or Gemini API key)
- Weight tracking with an SVG trend chart and Renpho CSV import
- Configurable daily targets, weight goals, and full data export/import as JSON

## Install as an app
Open the live link above in Chrome (Android) and choose "Install app" from the menu, or use the install prompt if it appears automatically.
